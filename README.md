# remote-web

> Information
----------

Upload sxc.php kedalam server target dan jalankan di remote.py <target.com/sxc.php> atau <target.com/[path]sxc.php>

Instal
--------

* apt-get update && apt-get upgrade
* apt-get install python2
* pip2 install requests
